# Как обновлять базу знаний бота

Бот читает информацию из папки `knowledge/`. Чтобы не править одно и то же в нескольких местах, **цены и программы хранятся в одном файле**.

## Что редактирует заказчик

| Файл | Что менять |
|---|---|
| `catalog.yaml` | Цены, названия курсов, описания, пакеты, рассрочка |
| `faq.md` | Гарантии, оплата, возврат, общие вопросы **без цен** |

**Не редактируйте** папку `generated/` — она собирается автоматически.

## Как изменить цену курса

1. Откройте `catalog.yaml`.
2. Найдите нужный курс в разделе `programs`.
3. Измените поле `price_rub` (число без пробелов, например `2990`).
4. Если меняете цену пакета — обновите `price_rub` в разделе `packages`.
5. Запустите сборку:
   ```
   python scripts/build_knowledge.py
   ```
6. Отправьте изменения в репозиторий (или передайте разработчику).

На сервере автоматически запустится сборка и обновление индекса. Если обновляете локально — после сборки выполните:
```
python scripts/reindex_knowledge.py
```

После этого бот:
- покажет новую цену в рекомендации курса;
- ответит новой ценой на вопросы клиентов;
- обновит FAQ и пакеты автоматически.

## Как добавить новый курс

1. Добавьте блок в `catalog.yaml` в раздел `programs` по образцу существующих.
2. Заполните обязательные поля: `id`, `level`, `title`, `price_rub`, `audience`, `result`, `summary`, `explanation`.
3. Запустите `python scripts/build_knowledge.py`.
4. Согласуйте с разработчиком правила рекомендации (какому уровню клиента подходит новый курс).

## Как изменить гарантии, возврат, оплату

Редактируйте `faq.md` — там только общая информация без цен.

## Пример: изменить цену «Финансовая база с нуля»

Было:
```yaml
price_rub: 2990
```

Стало:
```yaml
price_rub: 3490
```

Запуск:
```
python scripts/build_knowledge.py
```

Готово — больше ничего менять не нужно.

## Если нет доступа к командной строке

Передайте разработчику обновлённый `catalog.yaml`. Разработчик запустит сборку и выкатит на сервер.

## Структура папки

```
knowledge/
├── catalog.yaml      ← главный файл: курсы и пакеты
├── faq.md            ← гарантии, оплата, общие FAQ
├── README.md         ← эта инструкция
└── generated/        ← создаётся автоматически, не трогать
    ├── products.txt
    ├── catalog.md
    ├── packages.md
    └── price_faq.md
```
