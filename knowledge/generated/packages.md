## Пакеты из нескольких курсов

Школа предлагает готовые пакеты со скидкой при единовременной оплате.

Пакет «Финансовый фундамент» — курсы «Финансовая база с нуля» + «Личный бюджет и первая дельта». Полная стоимость по отдельности: 7990 рублей. Цена пакета: 7000 рублей. Экономия: 990 рублей. Срок доступа: по условиям каждого курса (6 и 9 месяцев соответственно) Для кого: Тем, кто хочет пройти путь от хаоса в тратах до стабильной дельты

Пакет «Путь инвестора» — курсы «Личный бюджет и первая дельта» + «Первые инвестиции». Полная стоимость по отдельности: 15000 рублей. Цена пакета: 13000 рублей. Экономия: 2000 рублей. Для кого: Тем, у кого уже есть база, и кто хочет перейти к первым вложениям

Пакет «Полная траектория» — курсы «Финансовая база с нуля» + «Личный бюджет и первая дельта» + «Первые инвестиции» + «Капитал под цели». Полная стоимость по отдельности: 32990 рублей. Цена пакета: 28000 рублей. Экономия: 4990 рублей. Срок прохождения рекомендуемый: 6–9 месяцев Для кого: Тем, кто хочет пройти весь путь от учёта до персональной стратегии капитала

Пакет «Капитал + сопровождение» — курсы «Первые инвестиции» + «Капитал под цели». Полная стоимость по отдельности: 25000 рублей. Цена пакета: 22000 рублей. Экономия: 3000 рублей. Для кого: Тем, у кого уже есть накопления и нужен переход к структурированной долгосрочной стратегии

Для пакетов от 10000 рублей доступна беспроцентная рассрочка на 3 месяца через партнёрские платёжные сервисы. Точные условия уточняйте у менеджера на момент оплаты.
